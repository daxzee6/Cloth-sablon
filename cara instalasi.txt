instalasi website
1.extrak file zip pengantar-desa
2. pindahkan folder pengantar-desa ke localhost htdocs
3. buka applikasi xampp
4. buka localhost/phpmyadmin
5. buat database baru dengan nama db_pengantar_desa_ci
6. buka mozila/chrome/opera pada laptop 
7. tuliskan pada website localhost/pengantar-desa
